Autoenconder

Autoencoders (AEs) are neural network models that learn the normal distribution of data by compressing input data into a low-dimensional representation and then reconstruct it. The encoder produces a latent space which is a set of fixed nodes that maps the input features to a low-dimensional speace. Optimization for accurate reconstruction of typical patterns allows the models to capture data's distribution and structural patterns. When new inputs deviate from the learned patterns, the higher reconstruction error indicates potential anomalies.

def count_stat(vector):
    # Because it is '0' and '1',
    # we can run a count statistic.
    unique, counts = np.unique(vector, return_counts=True)
    return dict(zip(unique.tolist(), counts.tolist()))

def descriptive_stat_threshold(df, pred_score, threshold):
    # Let's see how many '0's and '1's.
    df = pd.DataFrame(df)
    df.columns = feature_list
    df['Anomaly_Score'] = pred_score
    df['Group'] = np.where(df['Anomaly_Score']< threshold,
                 'Normal', 'Outlier')
    # Now let's show the summary statistics:
    cnt = df.groupby('Group')['Anomaly_Score'].count().reset_index().rename(columns={'Anomaly_Score':'Count'})
    # The count and count %
    cnt['Count %'] = (cnt['Count'] / cnt['Count'].sum()) * 100
    # The average
    stat = df.groupby('Group').mean().round(2).reset_index()
    # Put the count and the average together
    stat = cnt.merge(stat, left_on='Group', right_on='Group')
    return (stat)

def confusion_matrix_threshold(
    actual,score, threshold):
    Actual_pred = pd.DataFrame({'Actual': actual, 'Pred': score})
    Actual_pred['Pred'] = np.where(Actual_pred['Pred']<=threshold,0,1)
    cm = pd.crosstab(Actual_pred['Actual'],Actual_pred['Pred'])
    return (cm)

Ensemble Autoenconder (with Averages)

An ensemble autoencoder combines the outputs of multiple autoencoders, trained with various architectures, random seeds, or feature subsets, to produce a single, more reliable anomaly score. Each base autoencoder learns to reconstruct normal patterns; their reconstruction errors are then aggregated to form a consensus view of abnormality. This ensemble design reduces the variance and bias of any single model, captures structure in the data, and is more robust to noise in specific features or hospitals, producing an overal more realiable model.

Variational Autoencoders

Variational Autoencoders (VAEs) are a machine learning technique used to learn the underlying distribution of normal data by compressing and reconstructing it. By encoding input data into a lower-dimensional latent space and then decoding it back, the model learns to represent typical structures and relationships within the dataset. Unlike traditional Autoencoders (AEs), VAEs construct the latent space using two layers that represent the mean and the logarithm of the variance of a probability distribution, combined with a small amount of random noise. As a result, each reconstruction is slightly different, enabling the model to explore the entire latent space and generate diverse outputs. This probabilistic framework allows the VAE to capture uncertainty and variation in the data, making it particularly effective for detecting subtle or complex irregularities, where deviations from learned patterns manifest as higher reconstruction errors, indicating potential anomalies.
