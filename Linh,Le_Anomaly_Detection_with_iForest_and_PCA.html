iForest for Anamoly Detection in the Dataset

Isolation Forest is an unsupervised anomaly detection algorithm that isolates observations by randomly selecting a feature and a split value. The key idea is that anomalies are detected since they have shorter average path lengths in the trees. Hence, the model computes an anomaly score inversely related to the path length. Because the algorithms include several hyperparameters that influence the results, multiple models with different configurations will be trained and their results aggregated to improve robustness and stability. With the large dataset, a 5% anamolies might be too high. Hence, the analysis later defines threshold at 0.5%.

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

feature_list = ['Agency Number', 'Agency Name', 'Cardholder Last Name', 'Amount', 'Merchant Category Code (MCC)']     

def count_stat(vector):
    """Count how many 0s and 1s (Normal vs Outlier)."""
    unique, counts = np.unique(vector, return_counts=True)
    return dict(zip(unique, counts))

def descriptive_stat_threshold(df, pred_score, threshold):
    """
    Adds anomaly scores and labels (Normal/Outlier),
    then summarizes by group.
    """
    df = pd.DataFrame(df.copy()) 
    df.columns = feature_list     
    
    df['Anomaly_Score'] = pred_score
    df['Group'] = np.where(df['Anomaly_Score'] < threshold, 'Normal', 'Outlier')
    
    cnt = (
        df.groupby('Group')['Anomaly_Score']
        .count()
        .reset_index()
        .rename(columns={'Anomaly_Score': 'Count'})
    )
    cnt['Count %'] = (cnt['Count'] / cnt['Count'].sum()) * 100

    stat = df.groupby('Group').mean(numeric_only=True).round(2).reset_index()
    stat = cnt.merge(stat, on='Group', how='left')
    return stat

def confusion_matrix_threshold(actual, score, threshold):
    """Binary confusion matrix based on threshold cutoff."""
    Actual_pred = pd.DataFrame({'Actual': actual, 'Pred': score})
    Actual_pred['Pred'] = np.where(Actual_pred['Pred'] <= threshold, 0, 1)
    cm = pd.crosstab(Actual_pred['Actual'], Actual_pred['Pred'])
    return cm

def confusion_matrix(actual, pred):
    """Standard confusion matrix between actual and predicted labels."""
    Actual_pred = pd.DataFrame({'Actual': actual, 'Pred': pred})
    cm = pd.crosstab(Actual_pred['Actual'], Actual_pred['Pred'])
    return cm

import numpy as np
import pandas as pd


def top_outliers(df, scores, cutoff=0.5, model_name="IForest_Avg"):
    """
    Identify top outliers based on (possibly averaged) anomaly scores.

    Parameters
    ----------
    df : pd.DataFrame
        Original dataset (e.g., X_test)
    scores : np.array
        Anomaly scores from averaged model results (e.g., y_test_iforest_avg)
    cutoff : float
        Percentage of top anomalies to keep (e.g., 0.5 = top 0.5%)
    model_name : str
        Optional name prefix for output columns (e.g., "IForest_Avg" or "PCA")

    Returns
    -------
    outliers_df : pd.DataFrame
        Subset of df containing only the top anomalies,
        with '{model_name}_Score' and '{model_name}_Label' columns.
    """

    scores = np.asarray(scores).ravel()

    if not isinstance(df, pd.DataFrame):
        df = pd.DataFrame(df)

    threshold = np.percentile(scores, 100 - cutoff)

    df = df.copy()
    df[f"{model_name}_Score"] = scores
    df[f"{model_name}_Label"] = (scores >= threshold).astype(int)

    outliers = df[df[f"{model_name}_Label"] == 1].sort_values(f"{model_name}_Score", ascending=False)

    print(f" Model: {model_name}")
    print(f"Top {cutoff}% anomalies: {len(outliers)} out of {len(df)} total records")
    print(f"Threshold ({model_name}_Score): {threshold:.6f}\n")

    return outliers

PCA for Anamoly Detection in the dataset

Principal Component Analysis (PCA) is a statistical technique used to reduce the dimensionality of data by representing many variables with a smaller number of components. PCA groups together variables that show similar patterns, typically by measuring their correlations. Each principal component shows a direction of maximum variance in the data. Points that align closely with a given component are strongly correlated with that component. Anomalous points can then be identified as those that are farther away from the principal component axis. With the large dataset, a 5% anamolies might be too high. Hence, the analysis later defines threshold at 0.5%.
